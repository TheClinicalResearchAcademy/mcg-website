# Monache Consulting Group - Clinical Research Services Website

A responsive pharmaceutical clinical research services marketing website built with React, TypeScript, and Tailwind CSS.

## Features

- **Responsive Design**: Optimized for all devices with mobile-first approach
- **Pharma-Compliant Design**: Deep Teal (#0F766E), Cobalt Blue (#1E40AF), and Emerald (#059669) color scheme
- **Lead Generation**: Comprehensive forms for site registration and contact
- **File Upload**: Support for SOPs and training certificates upload
- **Trust Elements**: GCP, 21 CFR Part 11, and GDPR compliance badges
- **SEO Optimized**: Meta tags, OpenGraph, and semantic HTML
- **Accessibility**: WCAG AA compliant with proper landmarks and focus states

## Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS
- **Backend**: Express.js, Node.js
- **Forms**: React Hook Form with Zod validation
- **UI Components**: Shadcn UI
- **File Upload**: Uppy with object storage integration
- **Routing**: Wouter
- **State Management**: TanStack Query
